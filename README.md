# dotfiles

## Run the setup
```sh
$ sudo apt-get install -y curl
$ curl -fsSL https://github.com/swordf1zh/dotfiles/raw/master/setup.sh | sudo bash
```

## Setup Bash only
```sh
$ sudo apt-get install -y curl
$ curl -fsSL https://github.com/swordf1zh/dotfiles/raw/master/setup_bash_only.sh | bash
```
